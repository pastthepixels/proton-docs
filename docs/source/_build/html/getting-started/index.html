Getting Started With Proton¶

It’s quite simple to start a game in Proton - that’s one of its main goals. Thoughout this tutorial, I’ll show you how to set up a local game using Proton and Electron (I swear the naming is a coincidence), set up models in Blender, and create a small game where you can walk around.
