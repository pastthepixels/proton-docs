Proton3DScene¶

class Proton3DScene()¶

Description.

The constructor doesn’t have anything important; everything is done automatically.

Proton3DScene.camera¶

The scene’s camera (a Proton3DObject, created after initializing the scene)

Proton3DScene.children¶

An array consisting of the objects in the scene.

Proton3DScene.crosshair¶

A crosshair created after the function setCameraControls is executed. It has a property called “position”, which is helpful to see the point in which the player is facing.

Proton3DScene.extraFunctions¶

Functions that should be executed every now and then (close to whenever each frame is rendered, but not quite.)

Proton3DScene.extraKeyControls¶

An array of extra functions to be ran whenever a(ny) key is pressed.

Proton3DScene.interpreter¶

A Proton3DInterpreter class.

Proton3DScene.keys¶

An object consisting of the current keys being pressed at the current moment.

Proton3DScene.mappedKeys¶

An object consisting of key codes and their corresponding functions.

Proton3DScene.priorityExtraFunctions¶

An array of functions to be executed whenever a frame is rendered.

Proton3DScene.sky¶

A sky object that is created after initializing the scene.

Proton3DScene.getObjectList()¶

Gets the list of all the objects in the scene. You can also use [scene].children.

Returns

array – The array containing all the objects in the scene.

Proton3DScene.init(extras)¶

Initializes the scene. This is done automatically; you should instead see Proton.init() for user-configurable options.

Arguments

  • extras (object) – Parameters for initializing the scene.

Proton3DScene.pickUpObject(child)¶

“Picks up” an object.

Arguments

  • child (object) – The object to be picked up.

Proton3DScene.remove(parameter)¶

Removes an object from the scene.

Arguments

  • parameter (object) – The object to be removed from the scene.

Proton3DScene.resetPickingUp(child, scene, callback)¶

An automatic function called to stop picking up an object.

Arguments

  • child (object) – The object that is currently being picked up.

  • scene (object) – The scene for which it should be removed.

  • callback (function) – An optional function to be called once the object has been “let go”.

Proton3DScene.setCameraControls(extras)¶

Initializes mouse controls.

Arguments

  • extras (object) – The parameters for this function. An example: { distance: Proton.Vector3, xSensitivity: Number, ySensitivity: Number, invisibleParent: Boolean, cameraParent: Proton3DObject, type: “firstperson”|”thirdperson”, distance: Proton.Vector3 (for third person camera control types only), gun: Proton3DObject, gunPosition: Proton.Vector3 }

Proton3DScene.setKeyControls(obj, movementSpeed=2.5, jumpHeight, extras)¶

Initializes keyboard controls for the scene

Arguments

  • obj (object) – The player.

  • movementSpeed (number) – The movement speed of the player (optional).

  • jumpHeight (number) – The jump height of the player.

  • extras (object) – Optional extra parameters for this function. (You don’t have to worry about this as a user since it is automatic.)

Proton3DScene.setPickingUpControls()¶

Sets controls for using objects.

Proton3DScene.update()¶

Updates the scene: renders the scene, updates physics, and calls any functions under [scene].priorityExtraFunctions. Every other frame, it runs functions under [scene].extraFunctions.
