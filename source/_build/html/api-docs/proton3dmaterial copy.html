Proton3DMaterial¶

class Proton3DMaterial(parentObject, extras)¶

The class that represents a material for a mesh in Proton.

The constructor does some fancy stuff to work for imported objects, but you don’t have to worry about this since it is automatic.

Arguments

  • parentObject (object) – The (Proton3D)object that will have the material assigned to it

  • extras (object) – A list of parameters for this function. All that matters for this function is extras.name, which is the name of the material.

Proton3DMaterial.color¶

The color of the material, in a hexadecimal color value.

Proton3DMaterial.emissive¶

The emissive strength of the material: a value between 0 and 1.

Proton3DMaterial.emissiveColor¶

The emissive color of the material, in a hexadecimal color value.

Proton3DMaterial.metalness¶

The “metalness” of the material: a value between 0 and 1.

Proton3DMaterial.opacity¶

The opacity of the material: a value between 0 and 1.

Proton3DMaterial.roughness¶

The roughness of the material: a value between 0 and 1.

Proton3DMaterial.wireframe¶

Whether or not the material is rendered in “wireframe” mode.
